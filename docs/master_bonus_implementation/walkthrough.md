# 実装完了報告（Walkthrough）: マスター時永続ステータスボーナスの実装

## 概要
これまでインフレ防止のため廃止されていた「マスターボーナス（永続パッシブ）」の仕様を調整し、各職業の `job_classes` テーブルに設定されている `bonus_xxx` の値をそのまま用いて復活させました。

## 変更内容
1. **マスターボーナスの再構築**
   - `JobService::calculateFinalStats()` を修正し、キャラクターがマスター（★10）している職業をすべて取得するようにしました。
   - 取得したマスター済み職業の `bonus_hp` や `bonus_str` などの設定値を合計し、キャラクターの**「素のステータス（ベースステータス）」に加算**する処理を追加しました。

2. **表示上の変化と影響**
   - これにより、職業をマスターすればするほど、装備品を外しても基礎ステータス（画面上の白色の数字）が底上げされた状態になります。
   - 先ほど実装した「現在の職業に対するボーナス（緑色の数字）」とは別に適用されるため、「素の強さ」として視覚的に実感できるようになっています。

## 確認事項
- この機能は、現在データベースに設定されている `bonus_` 系カラムの値をそのまま利用するため、追加のデータ設定などは不要です。
- 剣士（ATKボーナス20）や僧侶（SPRボーナス20）などをマスターすれば、それぞれ基礎ステータスの ATK や SPR が永続的に+20されます。
- すべての職業をマスターした歴戦のプレイヤーは、強力な基礎ステータスを持った最強のキャラクターとして君臨することができます。
