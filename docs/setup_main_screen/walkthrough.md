# メイン画面初期セットアップ Walkthrough

## 実行内容
1. **Laravel 11のインストール**:
   - `c:\Users\yuta\tool\tool\ffa` ディレクトリにLaravel 11の新規プロジェクトをセットアップしました。
2. **パッケージの導入**:
   - **Livewire 3**: 画面遷移を伴わない非同期UIのためにインストールし、レイアウトにディレクティブを組み込みました。
   - **Tailwind CSS**: ユーティリティファーストなスタイリングのため導入し、ViteおよびPostCSS（v4プラグイン）経由でビルドが通るように設定しました。
3. **マイグレーションの作成**:
   - `characters` テーブルを作成しました（AP動的計算用の `last_action_at` や、検索・マッチング用途のインデックス付き）。
   - SQLiteデータベースでローカルマイグレーションを実行し成功しました。
4. **メイン画面（Livewireコンポーネント）の実装**:
   - `App\Livewire\MainScreen` クラスおよびビューを作成。
   - 2カラムレイアウトで、左側にステータスパネル、右側にタブ切り替え式のコンテンツエリアを実装しました。

## 次のステップ・動作確認方法
ローカル環境で以下のコマンドを実行し、ブラウザ（`http://localhost:8000` または出力されたURL）にアクセスしてUIを確認してください。

```bash
php artisan serve
```

> [!NOTE]
> 現在のステータスパネルの数値や画像はモックアップ（ダミーデータ）です。今後のステップで、データベースのレコードを実際に読み込んで表示する機能や、認証（ログイン）機能と紐付ける作業を進めていきます。
