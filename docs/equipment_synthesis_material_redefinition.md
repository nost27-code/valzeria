# 武具合成素材の再定義案

## 目的

市場実装に備えて、通常ドロップ素材そのものに需要を作る。

現行の進化合成は、同名装備、Gold、装備の欠片/進化石、都市素材、高位素材が中心になっている。  
通常敵が落とす350種の素材は、売却、ヴァルモン餌、素材交換所での変換には使えるが、武具合成の主要需要にはなっていない。

この再定義では、装備の欠片系統を必須素材から外しても成立するようにし、以下を主役にする。

- 種族共通素材
- 地域/都市素材
- 武器/防具/装飾品カテゴリ素材
- 上位敵/レア敵素材
- ボス固有キー素材

## 基本方針

### ボス固有素材は消費しない

`boss_unique` 素材は、合成の消費素材にしない。

用途は以下に限定する。

- レシピ解放キー
- 称号/プロフィール/図鑑表示
- 所持判定による施設・派生合成の解放

例:

- `番人の黒骨片` を持っていると、洞窟系装備レシピが解放される。
- 合成時に `番人の黒骨片` は消費しない。

### 装備の欠片系統は必須から外す

`装備の欠片`、`上質な装備の欠片`、`強装備の欠片` は、主役から外す。

扱いは以下のどれかにする。

1. 完全に合成必須素材から外す
2. 素材交換所で不足素材へ変換する救済素材にする
3. イベント/秘境系の補助素材に限定する

推奨は 2。  
合成そのものは通常素材で進め、欠片は不足分を補う汎用交換券のように扱う。

## 合成の基本式

各合成は以下で成立させる。

```text
進化元装備 x1
+ Gold
+ 種族/系統素材
+ 地域/都市素材
+ 装備カテゴリ素材
+ 必要なら上位/レア素材
+ ボス固有キー所持判定
```

## ランク別要求量

Goldは既存の `config/gold.php` の進化費用を使用する。

| 進化 | 素材要求の方針 | 消費素材量目安 |
|---|---|---:|
| G -> F | チュートリアル帯。種族共通素材だけ | 3から5 |
| F -> E | 種族共通 + 地域素材少量 | 5から8 |
| E -> D | 種族共通 + 地域素材 + カテゴリ素材 | 8から12 |
| D -> C | 地域素材の比重を増やす | 12から18 |
| C -> B | カテゴリ素材 + 地域素材 + レア素材少量 | 18から25 |
| B -> A | 上位通常素材/レア敵素材を要求 | 25から40 |
| A -> S | ボスキー所持 + 都市高位素材 + 上位素材 | 40から60 |
| S -> SS | 秘境/高難度素材 + 都市高位素材 | 60から90 |
| SS -> SSS | 秘境主/深部素材 + 高位素材 | 90から130 |
| SSS -> EPIC | 特別キー所持 + 最高位素材 | 130以上 |

| 武器傾向 | 主素材 | 補助素材 | 用途イメージ |

| 短剣/弓/銃 | 薄い翼膜 | 獣の毛皮、粗末な弓弦、歯車片 | 速度/技術武器 |
| 杖/魔導書 | 魔物の魔核 | 妖精粉、黒結晶 | 魔法武器 |
| 格闘 | 獣牙 | 獣の毛皮、魔物の外殻 | 素早さ/近接武器 |
| 聖剣/祈祷系 | 聖片 | 妖精粉、ボスキー所持 | 光/回復寄り武器 |
| 闇/魔王系 | 黒結晶 | 魔物の魔核、深淵素材 | 闇/終盤武器 |

## レシピ解放ルール

### G -> A

基本的に、該当地域へ到達していれば合成可能。

ただし、都市の最終ダンジョン装備や特殊装備はボスキー所持を要求する。

### A -> S

都市7ボス、または対応する節目ボスの `boss_unique` キー所持を要求する。

例:

```text
聖剣アークレア A -> S
解放: 教官の武勲章を所持
消費: アークレアの粗素材 x40 + 聖片 x20 + 魔鉱片 x20 + Gold
```

### S以上

ボスキーに加えて、秘境/深部素材を要求する。

消費素材は再入手可能なものだけにする。

## 装備の欠片系統の扱い

装備の欠片系統を完全削除する必要はない。
ただし、必須素材にしない。

推奨する扱い:

- 素材交換所で `装備の欠片 -> 任意の低級共通素材` に交換できる
- `上質な装備の欠片 -> 中級共通素材`
- `強装備の欠片 -> 高級共通素材`
- 秘境やイベントでの救済素材として残す

これにより、欠片を拾っても無駄にはならず、しかし市場需要の中心は通常素材に移る。

## 実装方針

段階的に進める。

### 第1段階

- 共通素材マスタを追加する
- 既存の350種通常素材を共通素材へ移行する
- 旧素材の通常ドロップを停止する
- `material_drops` を新素材に付け替える

実装済み:

- `common_drop` / `regional_drop` の共通素材30種を追加
- 旧 `MAT0001` 形式の通常素材を `legacy_normal_drop` として残し、所持数とドロップを共通素材へ統合
- `EnemyDropsSeeder` 側でも旧通常素材を共通素材へ読み替え、デプロイ時に旧素材ドロップが復活しないようにした
- `boss_unique` 素材は CSV 外の特別ドロップとして、Seeder 後も初回確定100%を維持する

### 第2段階

- G -> A の合成素材を再定義する
- 装備の欠片系統を必須から外す
- Gold + 共通素材 + 地域素材に寄せる

### 第3段階

- A -> S 以上にボスキー所持判定を導入する
- 消費素材は再入手可能素材だけにする
- ボスキーは消費しない

### 第4段階

- 市場の出品可能素材を共通素材/地域素材/レア素材に限定する
- `boss_unique`、討伐証、進化証、課金/支援素材は出品不可にする

## 注意点

- 既存所持素材の移行が必要。
- 旧素材を即削除せず、非アクティブ化または統合履歴を残す。
- 消費不可キーをレシピの `required_materials` に混ぜると誤消費リスクがあるため、解放判定として別扱いにする。
- 市場実装前に素材数を整理しておくと、出品UIと価格管理が大幅に楽になる。

## 結論

武具合成は、装備の欠片系統を主役から外し、通常ドロップ由来の共通素材と地域素材を消費する形へ再定義するのがよい。

ボス固有素材は消費しない解放キーにする。  
市場需要は、通常敵・上位敵・レア敵・秘境主から再入手できる素材に集中させる。
