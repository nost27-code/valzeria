## Stack

- App: Laravel 11 (PHP) + Livewire v3 + Blade + Alpine.js
- Styling: Tailwind CSS
- DB: MySQL (production on Xserver)
- Auth: Google OAuth, 1 account = 1 character
- Payment: Stripe (輝石 purchase; paid/free tracked separately)
- Tests: PHPUnit via `php artisan test`
- Deploy: `php local_deploy.php` (ZIP POST to valzeria.com); admin-only: `php local_deploy_admin.php`

## Product summary

ヴァルゼリアの冒険者 is a browser fantasy RPG recreating the feel of classic CGI-game FFA.
Core loop: login → explore/battle → EXP·Gold → level up → equip → job change → unlock next city → climb rankings.
Level cap: Lv255. Player-facing stat labels: HP / SP / ATK / DEF / MAG / SPR / SPD / LUK (old internal names like mp/str/agi remain in DB columns only).
Primary currencies: Gold (in-game), 輝石 (paid/support currency).
Main player entities: character (1 per user), jobs (rank ★1-10), equipment (with 銘 affixes), materials, valmon (companion), monster marks, arena rankings, tavern NPCs.
World: 10 cities (アークレア→…→ヴァルゼリア城), 40+ dungeons (area 1-70 normal, 71-74 special, 75-83 街道).

## Architecture notes

- Routing: routes/web.php; screens are Livewire components + Blade views
- Server pattern: thin Controllers, logic in app/Services/* (BattleService, ExplorationService, etc.)
- State management: Livewire component state + DB; no SPA framework
- DB access: Eloquent models (snake_case columns); DTO/BattleActor use camelCase
- Auth/session: Google OAuth login, character via Auth::user()->characters()->first()
- Logging: battle_logs, gold_transactions, kiseki_transactions, public_logs (bottom chat), admin analytics screens
- Battle: server-side auto turn-based; PRG pattern (redirect after POST); 3s cooldown via last_battle_at

- 鍛冶屋では武器・防具・装飾品を最大+5まで強化できる。武器・防具は従来どおり基礎性能+3%目安か最低+強化値分で伸び、装飾品は+1ごとに装飾品全体で合計+2の追加値を元の非ゼロ能力値比率で配分する。+1/+2は欠片中心、+3は石+欠片、+4は高純度石・石・共通素材・粗精錬核、+5は高純度石・石・共通素材・精錬核を要求する。鍛冶Goldは +1〜+5 の基本費用100G/500G/1,500G/5,000G/15,000Gに装備ランク倍率（G1.0、F1.2、E1.5、D2.0、C3.0、B5.0、A8.0、S15.0、SS30.0、SSS60.0、EPIC120.0）を掛ける。粗精錬核は素材交換所で織成核殻、晶糸核芯、魔物の欠片20個、5,000Gから作り、織成核殻は王都の織布、潮風の布片、精霊樹の繊維、黒鉄の装甲片を各10個、晶糸核芯は氷晶の織糸、砂金繊維、魔導繊維を各10個で作る。粗精錬核系の3レシピは素材不足でも素材交換所に表示する。素材交換所では敵固有素材を獣牙・小鬼の牙・獣の毛皮・魔鉱片などの共通素材へ圧縮でき、牙/爪/小鬼/ゴブリン系は汎用牙素材として獣牙へ寄せる。共通素材+100Gから強化石の欠片/守護石の欠片/調律石の欠片を合成できる。敵固有素材の共通化では、刻印、王印、神印、進化証、英雄の証、討伐証、導石、古代片、秘境晶、極印、ボス特異素材、進行キーを含む素材名は対象外にし、変換元素材名と交換先素材名が同じ自己変換レシピは作らない。黒結晶/呪い/闇/瘴気/腐、聖片/神殿/天使/司祭/聖、魔核/魔導炉/炉心/核は、鉱片/魔鉱/水晶/結晶/雷石/火種/鉱石より先に分類する。欠片20個+500Gを対応する石1個へ精製でき、高純度石は石5個+魔鉱片10個+2,000Gで精製する。精錬核は、粗精錬核1個、魔物の魔核20個、覇王黒晶、蒼炉魔晶、星樹氷晶を各1個、10,000Gで素材交換所から作る。覇王黒晶は王紋鋼・砂王金晶・ヴァルゼリア黒核を各5個、蒼炉魔晶は海鳴りの蒼鉱・炉心鋼・ルミナス魔晶・深魔骨核を各5個、星樹氷晶は精霊樹の琥珀・氷帝晶・セレスティア星晶を各5個で作る。素材交換所の受け取り素材名は詳細ページへ遷移せず、用途・必要素材・所持数をモーダルで表示する。敵ドロップの強化系補助枠では強化石の欠片・守護石の欠片・調律石の欠片だけを、各通常ダンジョンの約半数の通常敵から1%で許可し、石・高純度石・精錬核系素材は敵ドロップさせない。
- 装備の分岐進化S→SSは、各系統の古代片 x5 と素材交換所で作る複合素材 x3 を要求する。複合素材は聖天織晶、冥黒織晶、翠嵐織晶、鋼氷護晶、星導魔晶、風精織晶、砂海旅晶の7種で、都市素材・高位都市素材・地域素材を組み合わせて錬成する。聖天織晶は王都の守護布、冥黒織晶は瘴気の革片、翠嵐織晶は天空竜の織布も要求する。古代片の入手場所は現時点で未実装として表示し、未解放のS→SS装飾品は「現時点で未実装です」と案内する。武器・防具のSS→SSS分岐進化は、対応する秘境晶 x20 を要求する。装飾品のSS→SSS進化は、聖剣・魔剣・迅刃・重装・魔装・軽装・旅装の秘境晶を各2個ずつ使って素材交換所で作る原初装飾晶 x1 を要求し、裏ダンジョンボス撃破条件は要求しない。
