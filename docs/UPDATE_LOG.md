# UPDATE_LOG.md

Purpose: player-facing and operator-facing update notes for Valzeria.
This file records meaningful changes only.
Do not record tiny refactors, formatting-only changes, or AI docs-only edits.

## Unreleased

### 2026-06-28

#### Added
- 管理画面に統計分析を追加し、新規登録、活動者、戦闘、チャット、売上の日別推移と伸び率を確認できるようにしました。

#### Changed
- Aランク装備の通常獲得は全体チャットへ流さず、特別な装備獲得だけを知らせるようにしました。

#### Balance
- 通常探索やボス戦で、敵の攻撃を回避だけで受け流し続けにくいよう命中率を調整しました。
- 魔王城ヴァルゼリアの通常敵とボスを強化し、敵の攻撃を回避だけで突破しにくいよう調整しました。

#### Internal
- 管理画面の戦闘シミュレーションで、登録済みの敵を全件候補として選べるようにしました。

### 2026-06-27

#### Added
- 管理画面で下部チャットの各種ログを検索し、選択して削除できるようにしました。
- 管理画面でNPCごとの調達素材、在庫、出品中数量、販売済み数量と販売額を確認できるようにしました。

#### Changed
- 戦闘中や戦闘結果画面でも、ホーム画面下部と同じチャット・公開ログを確認できるようにしました。
- ダンジョンの目安戦力を、推奨Lvではなく実際に出る通常敵の戦力範囲から表示するようにしました。
- チャンプ戦でレベルの低い側ほど必殺技が発動しやすくなるようにしました。
- チャンプ戦で挑戦者とチャンピオンが現在職の必殺技を使うようにしました。
- 戦力の表示値を見直し、計算結果を10倍せずそのまま表示するようにしました。
- チャンプカードに現在チャンピオンの戦力を表示し、HP/SPバーも確認しやすくしました。
- スマホ上部のステータスバーに現在戦力を追加し、名前・レベル・職業・戦力の順で見やすくしました。
- 探索先の目安を推奨Lvから目安戦力に変更し、現在の戦力と比べやすくしました。
- 上部ステータスのLvと職業を縦並びにし、スマホでも職業名が読みやすい表示に調整しました。
- スマホ表示の管理画面メニューを、左上ボタンから開閉できるサイドバー形式に変更しました。
- 市場に素材を出品するNPCから、上級職・レジェンド枠のNPCを除外しました。
- 市場のNPC素材出品を専用枠ではなく、通常の買う一覧に混ぜて表示するようにしました。

#### Fixed
- 公開ログ管理で管理人メッセージが通常の選択削除に混ざらないよう、削除保護を追加しました。
- 未使用BPが残ったまま転職できてしまう問題を修正し、転職前にすべて能力へ割り振る必要があるようにしました。
- アクセサリー進化の秘境条件が街道解放で満たされないようにし、裏ダンジョン解放・撃破を正しく参照するよう修正しました。
- 魔法使いの奥義「魔力の火種」が、説明どおり小さな魔法攻撃と自身の魔力上昇を行うように修正しました。
- チャンプ戦の次回挑戦時刻が日本時間で表示されるように修正しました。

#### Balance
- 戦力計算でSP/MPの加点を控えめにし、素早さの評価が後半まで伸びるよう調整しました。
- チャンプが連勝するほど戦闘能力が少しずつ下がる「連勝疲労」を追加しました。
- 通常探索で敵から装備品がドロップする確率を下げ、装備更新ペースを調整しました。
- チャンプ戦で相手とのレベル差が大きい場合、まれに大きな追加ダメージを与えられるようにしました。
- チャンプ戦で勝敗に関わらず得られる素材報酬を1〜2個に調整しました。
- 砂都サンドラ以降の敵から得られるEXPの伸びを緩やかにし、後半のレベル上昇ペースを調整しました。

### 2026-06-26

#### Added
- 酒場で出会えるNPCや冒険者名簿に、キャラ画像を表示するようにしました。
- 敵ドロップ防具に、能力銘や種族耐性が付く銘付き防具システムを追加しました。
- 神殿の職業カードに詳細ボタンを追加し、特徴・伸びやすい能力・奥義・マスター恩恵・必要条件を確認できるようにしました。
- 敵ドロップ武器に、能力銘や種族特効が付く銘付き武器システムを追加しました。

#### Changed
- PC表示時の冒険者プロフィール内スクロールバーを、細く見やすいデザインに調整しました。
- 冒険者プロフィールの外周フレームを外し、表示をすっきりさせました。
- Lv30以上で現在の職業をマスターしているとき、次やることに転職案内を表示するようにしました。
- ヴァルモンに餌を与えた後も餌タブに残り、結果をトースト通知で確認できるようにしました。
- 倉庫がいっぱいの案内から、そのまま倉庫整理へ移動できるようにしました。
- 下部チャットの自動更新間隔を30秒に変更し、サーバー負荷を抑えました。
- 下部チャットを展開したとき、通常より15行多く履歴を見返せるようにしました。
- チャット送信先から未使用のギルドを外し、全体と個人だけに整理しました。
- 戦闘画面の敵情報で、属性等ではなく種族が分かるようにしました。
- 補給所に保管されている回復アイテムの内訳を、持ち越し分と本日分に分けて表示するようにしました。
- 素材の入手場所へ移動したとき、対象の探索先がより目立つようにしました。

#### Fixed
- ヴァルモン餌やり後のトースト通知が、アプリ追加バナーに隠れないように修正しました。
- 次やることリストで、到達済みの新しい街に未完了ダンジョンがあるのに前の街道が案内されることがある問題を修正しました。
- 深い階層へ進む画面の推奨Lv表示が、現在の深度設定に合うように修正されました。

#### Balance
- 職業マスター上限を全職ランク10に統一し、必要職業EXPを中級職2倍、上級職5倍、伝説職10倍に調整しました。
- 次の街へつながる道ダンジョンの敵EXPを、同じ強さ帯の通常敵に近づけました。
- 通常探索の待機時間を階層別に調整し、表層は10秒、深部は15秒、深層以降は20秒にしました。

#### Internal
- デプロイ時にLaravelの設定・イベント・Bladeテンプレートの本番キャッシュを生成するようにしました。
