# A-3 所属・冒険者協会 追記案（**2026-07-02 承認・DOMAIN_RULES/FEATURE_STATUSへ適用済み**）

裁定（2026-07-02）: 所属は今後の正式機能候補として採用。寄付ランク・救助費軽減は「停止中・未公開機能」扱い（削除も自動再開もしない）。銀行預金まで救助費対象にする仕様は非採用。ロスト・救助費の対象は原則所持Goldのみ。軽減率の数値変更はしない（コード現行値にドキュメントを合わせる）。

## 現行コードの確認結果（読み取りのみ・2026-07-02時点）

- 敗北時Goldロスト: **所持Gold（`characters.money`）の10%固定**（`GuildService::DEFEAT_GOLD_LOSS_RATE = 0.10`、`ExplorationService`:538 が `money` のみを渡して `GoldService::spend` で控除）。銀行預金・輝石・倉庫・市場出品中資産は対象外（コード上も参照していない）。
- 協会ランク表（一般10%〜伝説級後援者5%、fee_cap付き8段階）は `GuildService::RANKS` に実装済みだが、`calculateRescueFee` は **fee_cap=0・料率は10%固定で実質バイパス**しており、ランクによる軽減は現在無効。
- 寄付: `GuildService::donate()` は「冒険者協会の寄付機能は現在利用できません。」を返す停止状態。
- 戦利品（素材・アイテム）ロスト: 通常**50%**、救助保険証で**25%**、緊急救助要請で**0%**（`ExplorationService`:501-513）。
- ヴァルモン卵: 敗北時ロストあり。緊急救助要請時は保護（`ExplorationService`:516-519）。
