# Aランク装飾品の進化に必要な素材と入手場所

装飾品の「Aランクへの進化（B→A）」と「Aランクからの進化（A→S）」で必要な素材が異なります。データベースを解析した結果を以下にまとめます。

## 1. Bランク → Aランク への進化（Aランク装飾品の作成）

Bランクの装飾品をAランクに進化させる場合、ベースとなるBランク装飾品に加えて、**基礎素材（22個）＋ 黒鉄の部材（4個）＋ サブ素材（4個）** が要求されます。

### 代表的な必要素材
装飾品の種類によって以下のいずれかの組み合わせが求められます。

* **基礎素材（22個）**
  * `獣牙`、`古びた徽章`、`薄い翼膜`、`聖片` など
* **共通素材（4個）**
  * `黒鉄の部材`
* **サブ素材（4個）**
  * `魔鉱片`、`竜鱗`、`妖精粉`、`魔力水晶`、`魔物の欠片`、`獣の毛皮`、`羽根` など

### 素材の主な入手場所（ドロップ先）
データベースに登録されている主なドロップ対象モンスター・エリアです。

* **獣牙**: 氷牙ウルフ(35%), 草むらウルフ(45%), 丘ウルフ(45%), 野うさぎ(25%) ほか
* **薄い翼膜**: 草原コウモリ(28%), 海蝕コウモリ(35%), 闇コウモリ(35%) ほか
* **聖片**: 月光ピクシー(35%), 深海封印守(35%), 星見星読みの亡霊(35%) ほか
* **魔鉱片**: 海蝕ゴーレム(28%), 雪山ゴーレム(35%), 溶鉱炉の炎獣ブレイズマンティス(100%・ボス) ほか
* **竜鱗**: 氷竜の幼体(35%), 黒炎竜(28%), 竜王バハムート(100%・ボス) ほか
* **妖精粉**: 泉のフェアリー(35%), 泡の精霊(35%), 若葉フェアリー(28%) ほか
* **魔力水晶**: シーラット(35%), 浜辺クラブ(35%), 地下ワニ(35%) ほか
* **獣の毛皮**: 野うさぎ(35%), 丘ウルフ(35%), 若葉ウルフ(35%) ほか
* **羽根**: 天空鳥(35%), 神域祭壇天使(35%), 上層ハーピー(35%) ほか
* **黒鉄の部材** / **古びた徽章** / **魔物の欠片**: 現状のモンスタードロップや探索報酬のテーブルには紐づいておらず、店売りやイベント交換など別の手段で入手する可能性があります。

## 2. Aランク → Sランク への進化

Aランクの装飾品をさらにSランクへ進化させる場合、すべての装飾品で共通して「装飾の核×3」と、各装飾品の系統に応じた「結晶×8」が要求されます。

### 素材の主な入手場所
現在、`装飾の核` および 各種 `結晶` については、**敵からのドロップ・エリアドロップ・探索ルート報酬のいずれのデータベースにも入手先が登録されていません。** 

これらは未実装であるか、ショップでの購入、闘技場・ギルド等での交換、あるいは今後のアップデートで追加されるコンテンツ（高難易度ボスなど）の報酬となっている可能性が高いです。
