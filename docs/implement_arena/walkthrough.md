# 闘技場（ランク戦）機能の実装完了

ご指定いただいたルールに基づき、「1位を目指す順位制ランク戦」をメインとした新しい闘技場機能を実装いたしました！

## 実装した主な機能

### 1. 新しい順位・昇格システム
* **絶対順位制**: 闘技場に参加した冒険者全員で1位〜最下位までの順位を競います。
* **1ランクアップ制**: 上位の対戦相手（自分より1〜3つ上の順位のプレイヤー）に挑み、**勝利すると自分の順位が1つ上がります**。
  * 同時に、自分の1つ上にいたプレイヤーが1つ順位を下げ、入れ替わる形になります。
  * 例：10位のプレイヤーが上位（例えば7位）に勝った場合、自分は9位に昇格し、元々9位だったプレイヤーが10位に降格します。
* **敗北・防衛**: 敗北しても順位は下がりません。（※自分がログインしていない時に他のプレイヤーに下から突き上げられて抜かれた場合は、玉突きで順位が下がります）

### 2. リニューアルされた闘技場画面
* **現在の順位表示**: 画面右上に、現在の自分の順位と通算戦績（〇勝〇敗）を大きく表示します。
* **ランク戦に挑む**: 自分より「1〜3つ上位」のプレイヤーがリストアップされ、「挑む」ボタンから即座に対戦を開始できます。
* **対戦ログ（順位変動ログ）**: 自分が誰に挑んで勝った/負けたか、あるいは誰に挑まれて防衛した/抜かれたかの履歴を時系列で確認できます。
* **トップランカー**: サーバー内の上位3名（1位〜3位）のプレイヤーが常に表示され、目標として意識できるようになっています。

### 3. バトルシステム
* 探索（PvE）のバトルロジックをベースに、プレイヤーのステータスや職業スキルをそのまま反映する本格的なPvPバトルエンジン（`PvPBattleService`）を新規開発しました。
* 挑戦回数の制限やスタミナ消費はありませんので、何度でも編成やスキルを試しながら上位に挑むことができます。
