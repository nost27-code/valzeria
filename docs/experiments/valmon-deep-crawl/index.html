ヴァルモン地下探検とは

ヴァルモン地下探検は、ヴァルモンだけが入れる小さな地下道を進み、どこまで深く潜れるかを競う週末限定コンテンツです。

冒険者は中に入れません。地下へ進むのは、あなたの相棒ヴァルモンだけです。

シーズン中にカードを集め、デッキを組み、TP（探検ポイント）で強化しながら、より深い階層の突破を目指しましょう。

開催期間

ヴァルモン地下探検は、週末シーズンとして開催されます。

  • 開始：金曜 9:00
  • 終了：月曜 9:00

シーズンが終了すると、ランキングが確定します。

シーズン中に集めたカード、白風金貨、TP、能力強化、装備枠拡張、進行階層は次回シーズンに持ち越されません。

基本の流れ

  1. 金貨や階層報酬でカードを集める
  2. 所持カードからデッキを組む
  3. ヴァルモンを地下へ進ませる
  4. 階層を突破してTP（探検ポイント）を得る
  5. TPでカード取得・能力UP・装備枠拡張を行う
  6. さらに深い階層を目指す

最初はカードを1枚も持っていません。時間経過や階層突破によって、少しずつカードを増やしていきます。

金貨について

金貨は、カード取得やTP強化に使えるシーズン中の準備リソースです。

金貨は時間経過で解放されます。開始時に2枚受け取れ、その後は4時間ごとに2枚ずつ解放されます。

  • 開始時：2枚
  • 以後4時間ごと：2枚
  • 月曜5:00の解放で最大36枚

金貨は、次のどちらかに使えます。

  • 金貨1枚：カード3択を1回引く
  • 金貨1枚：シーズンTP +1

金貨は固定コストです。何回使っても、必要枚数は1枚です。

未使用の金貨はシーズン中残ります。使用済みの金貨は戻りません。

カードについて

カードは、ヴァルモンを強化するためのシーズン限定効果です。

カードには、攻撃、防御、回復、探知、回避などの種類があります。

カードを集める方法は主に3つあります。

  • 金貨を消費してランダムから選ばれた3択から1枚を引く
  • 1〜10階の初回クリア報酬でカードを得る
  • TPを消費してランダムから選ばれた3択から1枚を引く

カード3択では、表示された3枚の中から1枚を選びます。一度表示された3択候補は保存されます。画面を閉じたり、敗北したりしても、候補は引き直せません。

選んだカードは、そのシーズン中ずっと使用できます。

今週のカードプール

カードは全部で100枚あります。ただし、毎シーズンすべてのカードが出るわけではありません。

  • 今週出現するカード：80枚
  • 今週休眠するカード：20枚

休眠中のカードは、そのシーズンでは入手候補に出ません。

今週どのカードが出現し、どのカードが休眠しているかは、カード一覧から確認できます。

デッキについて

集めたカードは、デッキにセットすることで効果を発揮します。

初期装備枠は20枚です。所持カードが増えても、すべてを装備できるとは限りません。どのカードを入れるかが重要です。

装備枠は、TPを使って広げることができます。最大装備枠は32〜36枚を想定しています。

装備枠を広げるほど、多くのカード効果を使えるようになります。ただし、そのぶんTPを消費するため、能力UPやカード取得に使えるTPは少なくなります。

TPについて

TPは、探検ポイントの略です。そのシーズン中だけヴァルモンを強化するリソースです。

TPは、次の能力に振り分けられます。

  • 攻撃
  • 防御
  • 探知
  • 回避

TPは、地下へ降りた時に階層ごとに1回だけ得られます。1階層につき得られるTPは固定で、同じ階層から二重に受け取ることはできません。

TPは、カード3択、TP増加、装備枠拡張、能力値強化に使えます。

TPによる強化は敗北しても残ります。ただし、使用済みTPは戻らず、シーズン終了時にすべてリセットされます。

カード3択、TP増加などの通常TP消費は、使うたびに必要TPが増えていきます。

  • 1回目：1TP
  • 2回目：2TP
  • 3回目：3TP
  • N回目：N TP

たとえば、1回目にカード3択を引くと1TP、2回目にTPを増やすと2TP、3回目に能力を1上げると3TPが必要になります。通常TP消費回数は、敗北しても戻りません。

装備枠拡張のTP消費

装備枠拡張は、通常TP消費とは別のコストで管理されます。

  • 1回目：1TP
  • 2回目：3TP
  • 3回目：5TP
  • 4回目：7TP
  • N回目：2N - 1 TP

装備枠を多く広げるほど、強力なカードをたくさん積めるようになります。ただし、後半ほどコストが重くなるため、どこまで枠を広げるかが重要です。

階層突破について

「進む」を押すと、現在の階層に挑戦します。勝利すると次の階層へ進み、敗北するとその階層に留まります。

23階を突破すると次は24階へ進みます。24階で敗北した場合は、もう一度24階から挑戦できます。

敗北しても、1階からやり直しにはなりません。デッキを見直したり、金貨でカードを増やしたり、TPで強化したりして、同じ階層に再挑戦できます。

1〜10階の初回報酬

1階から10階までは、各階の初回クリア時にカード3択が発生します。この報酬は、各階につきシーズン中1回だけです。

一度クリアした階層では、再びカード報酬を受け取ることはできません。

敗北時に残るもの

敗北しても、現在階層、最高クリア階層、所持カード、未使用の金貨、TP、TPで得た能力強化、TPで広げた装備枠、デッキは残ります。

敗北しても失われないため、カードや強化を見直して再挑戦できます。

敗北時に戻らないもの

使用済みの金貨、使用済みのTP、カード3択の引き直し、クリア済み階層の初回報酬は戻りません。

一度使ったリソースや、一度表示されたカード候補はやり直せません。

ランキングについて

ランキングは、シーズン中の最高クリア階層で競います。より深い階層を突破したプレイヤーほど上位になり、同じ階層の場合はスコアで順位決定します。

スコアは、階層突破、残HP、選んだルートや探知カードの効果などで加算されます。シーズン終了時にランキングが確定します。

課金による金貨の先取り

課金によって、金貨を先取りできる場合があります。ただし、課金で金貨の最大数が増えるわけではありません。

  • 無課金でも最大36枚
  • 課金しても最大36枚
  • 課金は未来分を先に受け取るだけ

課金による先取りで得た金貨も、通常の金貨と同じようにカード3択やTP増加に使います。

TPの購入、装備枠の直接購入、ランキング挑戦回数の購入、敗北時復活などはありません。

攻略の考え方

カードを多く集めれば、デッキの選択肢が増えます。能力を上げれば、カード運に頼らず安定しやすくなります。装備枠を広げれば、より多くのカード効果を同時に使えます。

ただし、すべてを同時に伸ばすことはできません。カードを引くか、能力を上げるか、装備枠を広げるか。金貨とTPをどう使うかが、地下探検の鍵です。

重要な注意

このコンテンツのカード、金貨、TP、能力強化、装備枠拡張は、すべてシーズン限定です。

シーズン終了後、次回シーズンでは再びカード0枚から始まります。毎シーズン、出現カードや休眠カードが変わるため、同じ攻略法がいつも通用するとは限りません。
