# 職業システム拡張計画（全43職体制への移行）

## 概要
ユーザーからご提案いただいた「王道ルート」「商人ルート」「魔法ルート」などの成長ツリーの多様性を持たせた**全43職業（基本8、中級18、上級12、伝説5）**への大規模拡張を実施します。

### 1. `JobSystemSeeder.php` の大改修
提案いただいた表の内容に沿って、すべての職業データ、ステータス倍率、転職条件（必要マスター職）を `JobSystemSeeder.php` に反映させます。

* **基本職 (8種)**: 既存のまま
* **中級職 (18種)**: 「侍」「軍師」「剣闘士」「守護騎士」「傭兵」「魔盗士」「モンク」「魔弓士」「吟遊詩人」「薬師」を新設・追加。
* **上級職 (12種)**: 「竜騎士」「武神」「幻影王」「機工王」「神官戦士」「影狩人」「賢商王」を新設・追加。
* **伝説職 (5種)**: 既存の3種に加え、「蒼竜王」「時空王」を追加。また、転職条件に指定された「上級職3種のマスター」を `requirements` に組み込みます。
* **マスターボーナス**: 新設された職業に対しても、その職の特徴を活かしたマスターボーナス（`hp_rate`, `atk_rate` などの上昇）をシードに追加します。

### 2. `SkillSeeder.php` への追加職業のスキル紐付け
職業が増加することに伴い、新設された職業に対しても戦闘用の必殺技（スキル）を割り当てる必要があります。
現時点では、**先ほど作成した26種類のスキルのうち、最もイメージに合うものを再利用して割り当てます**。（後日、必要に応じて専用の必殺技を作成します）

**【暫定スキル割り当て案（例）】**
- **侍**: 精密射撃（抜刀術のイメージ）
- **剣闘士**: 全力斬り
- **守護騎士**: シールドスマイト
- **傭兵**: 全力斬り
- **モンク**: 三連撃
- **魔弓士**: スナイプショット
- **竜騎士**: ブレイブブレイド または スナイプショット
- **蒼竜王**: アビスバースト
- **時空王**: グランドクロス

> [!IMPORTANT]
> **オープンクエスチョン**
> 1. 新しく追加される職業に対して、とりあえずは「既存スキルの流用」で紐付けてよろしいでしょうか？（後々、また各職業に専用スキル名をつけることも可能です）
> 2. 伝説職の「特定ボス撃破」「専用アイテム」といった追加条件はシステムが未実装のため、おっしゃる通り今回は `is_hidden = true` ＋ 「上級職3つのマスター」で実装し、後ほど（新しいダンジョン実装時などに）ボス撃破の条件フラグなどを追加していく方針で進めてよろしいでしょうか？

承認いただけましたら、この構成で開発（Seederの実装と反映）に進みます！
