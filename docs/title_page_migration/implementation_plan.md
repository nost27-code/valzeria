# 称号一覧ページの独立化・タップ装備実装計画

## 概要
現在モーダルとして表示している称号一覧を独立した専用ページ（`/titles`）に変更し、一覧性と操作性を向上させます。
また、ユーザーのご要望に合わせて「称号のパネル自体をタップするだけで装備が切り替わる」ようにUIを改善します。

## 実装内容

### 1. 新規ルーティングの追加
- `routes/web.php` に `/titles` （称号一覧画面）へのルートを追加します。
- `LeftSidebar` の「称号」エリアをクリックした時の挙動を、モーダルを開く処理から `/titles` への画面遷移に変更します。

### 2. Livewireコンポーネントの移行・新設
- 現在のモーダル用コンポーネント `TitleListModal.php` を廃止（またはリネーム）し、フルページコンポーネント `App\Livewire\TitleList.php` として再実装します。
- レイアウトは既存の `components/layouts/app.blade.php` に則り、左にステータス（LeftSidebar）、右に称号一覧という2カラム構成にします。

### 3. UI/UXの改善（パネルタップ装備対応）
- **タップ装備:** 小さな「装備する」ボタンを廃止し、**「未装備で取得済みの称号のパネル全体」をタップ可能（クリッカブル）** にします。
- **視覚的フィードバック:**
  - タップ可能なパネルにはホバー時に浮き上がるアニメーション（`hover:-translate-y-1` 等）や、カーソルをポインター（`cursor-pointer`）に変更するスタイルを適用します。
  - 「装備中」の称号は、背景色や枠線を明るい色（ゴールド等）でハイライトさせ、一目でわかるようにします。
  - 画面上部には「街に戻る」ボタンを配置し、メイン画面（ホーム）へ戻りやすくします。

## 検証計画
- 左サイドバーの称号部分から `/titles` へ正常に遷移できるか確認。
- 称号パネルをタップした際、非同期（Livewire）で即座に「装備中」状態に切り替わるか確認。
- 装備を切り替えた際、左サイドバーの称号表示が新しいものに即座に更新されるか確認。
