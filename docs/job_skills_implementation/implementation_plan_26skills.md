# 全26職業 固有必殺技（スキル）実装計画

ご要望の通り、すべての職業（全26職）にそれぞれ専用の必殺技名を与え、基礎となるアクション（連続攻撃、防御無視、ステータス依存など）は流用しつつ、威力や効果に差をつけて実装します。

## 1. アクションクラスの追加調整
既存の特殊なアクションに加え、シンプルに高威力な一撃を放つための標準アクションを追加します。
- `HeavyPhysicalAction.php`: 単発の高威力物理攻撃。
- `HeavyMagicalAction.php`: 単発の高威力魔法攻撃。
- さらに既存のアクション（連続斬りやみだれうちなど）内で、スキルの `power` や `accuracy` が反映されるように調整します。

## 3. 実装手順
1. `HeavyPhysicalAction` および `HeavyMagicalAction` クラスの新規作成。
2. 既存の `SkillSeeder.php` の内容を破棄し、上記の26種類のスキルをそれぞれデータベースに登録（威力やパラメータを微調整）。
3. 全26職業に対して、対応する固有スキルを `assignSkillToJob` で紐付け。
4. DBシードを実行（`php artisan db:seed --class=SkillSeeder`）。
