# 必殺技（スキル）追加実装計画

現在、必殺技は「連続斬り」「無明剣」「メテオ」の3つしかなく、戦士・格闘家・剣聖・魔法使い・大賢者（賢者）にしか設定されていません。
全26職業がそれぞれの特徴を活かして戦えるよう、新しい必殺技を追加し、全職業に割り当てます。

## 1. 新規追加する必殺技（スキル）案

以下の必殺技を新しくプログラム（`ActionClass`）として実装し、データベースに登録します。

### 物理系（ATK依存）
* **みだれうち**（対象: 盗賊、忍者、弓使いなど）
  - 効果: 対象に3〜4回の連続ダメージを与える。（連続斬りの強化版）
* **急所狙い**（対象: 弓使い、狙撃手）
  - 効果: 命中率は少し下がるが、必ずクリティカルヒット（大ダメージ）になる。
* **捨て身の攻撃**（対象: 狂戦士、暗黒騎士）
  - 効果: 自身も最大HPの10%ダメージを受けるが、通常の2.5倍の極大ダメージを与える。
* **シールドバッシュ**（対象: 聖騎士など）
  - 効果: 自分のATKではなく「自身のDEF（防御力）」を攻撃力に変換してダメージを与える。防御特化でも火力が出る。

### 魔法系（MAG・SPR依存）
* **ホーリー**（対象: 僧侶、司祭）
  - 効果: 自分のSPR（精神力）を魔法攻撃力として扱い、大ダメージを与える。
* **アルテマ**（対象: 大賢者、古代錬成王）
  - 効果: 相手のSPR（魔法防御）を半分無視してダメージを与える魔法版・無明剣。

### 特殊系
* **銭投げ（ゴールドラッシュ）**（対象: 商人系）
  - 効果: 運（LUK）とレベルに依存したランダム幅の大きな固定ダメージを与える。（所持金を消費する仕様だと処理が複雑化・オート戦闘で全滅するリスクがあるため、商人ならではの運任せダメージとする）
* **ギガノヴァ**（対象: 魔法剣士、勇者、ヴァルゼリアの英雄）
  - 効果: 自身の「ATK」と「MAG」を合算した数値でダメージ計算を行う究極の魔法剣。

## 3. 実装ステップ
1. ユーザー（あなた）に上記のスキル案と割り当てを確認・承認いただく。
2. 追加するスキルのアクションクラス（例: `MidareuchiAction.php`, `SutemiAction.php` など）を `app/Services/Battle/Skills` 配下に作成。
3. `SkillSeeder.php` を更新し、全職業に `assignSkillToJob` を用いて紐付ける。
4. `php artisan db:seed --class=SkillSeeder` で反映。

> [!NOTE]
> 「この職には別のスキル名がいい」「こういう効果の技が欲しい」といったご要望があれば、ぜひお教えください！

承認いただけましたら実装に入ります。
