## Progression

- Exploration: 通常探索の輝く宝箱は探索度100以上で抽選対象になる。探索度100到達は宝箱確定ではない。
- Battle: 通常探索の連戦待機は通常探索に適用する。表層は10秒、深部は15秒、深層/最深層/異界層は20秒。ボス挑戦と現在表示中の深部入口へ進む確認後の1戦は通常探索の連戦待機では止めないが、HP不足や宿屋直後の待機など開始条件は適用する。通常探索の深部/深層/最深層/異界層へ入場した時点で危険度を0%に戻し、入場済み階層は維持したまま次階層に向けて危険度を再蓄積する。深度目標Lvはエリア推奨Lvに対して深部+5、深層+15、最深層+25、異界層+40。深部以降の敵は、危険度倍率補正と深度目標Lvに対する種族・派生・役割ベースの敵ステータス生成式の高い方を戦闘時ステータスに使う。街道敵の通常EXPは、同Lv帯の通常敵に近い強さ相応のカーブで付与する。
- Drops: 通常戦の素材抽選は敵固有の素材ドロップを優先し、該当カテゴリがない場合は進行帯ごとのフォールバック素材プールから抽選する。地域素材カテゴリでは敵固有候補がある場合も都市の地域素材を低確率候補に残す。強化石・守護石系はこのフォールバックの enhance 枠にも含まれる。敵ドロップ武器のうち `affix_enabled=true` のものは、ドロップ確定時に銘なし70%/能力銘20%/能力銘+種族銘8%/良品1.5%/逸品0.5%で銘抽選を行う。銘補正は個体に保存され、種族特効はPvEの敵への直接ダメージにのみ適用し、PvP/チャンプ戦には適用しない。
- Valmon: 相棒ヴァルモンは通常探索勝利後にLv1で素材発見2%、Lv10で2.5%、Lv20で3%。Lv30から得意素材補正、Lv40から通常探索戦闘中1回だけ低確率追撃、Lv50から未発見要素の気配ヒント、Lv75からHP低下時の応急回復を1探索チェーン1回まで、Lv100で称号「名相棒」と特別ログ/プロフィール表示向け効果を解放する。
- Arena rank battle: 格上に勝った場合、勝者は相手の旧順位まで上がり、その間にいた冒険者は1つずつ下がる。格下に勝った場合、順位は変動しない。
- Jobs/class change: <short>
- Equipment: 鍛冶強化は最大+5。+1ごとに基礎性能+3%を目安に伸び、低い基礎値では最低+強化値分を加算する。必要素材は+1/+2/+3/+4/+5で1/3/5/7/9個。武器は強化石、防具は守護石、装飾品は装飾強化石を使う。素材交換所では各欠片3個を対応する石1個へ精製できる。

## Public logs

- Loggable events: ランク戦で勝利して闘技場順位が上がった場合は全体チャットに番付上昇ログを流し、11位以下から10位以内に入った場合はTOP10入りログも流す。管理者は管理画面から管理人メッセージを全体チャットへ投稿できる。
- Forbidden data: private user data, secrets, internal IDs unless safe.
