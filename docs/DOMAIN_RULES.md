## Progression

- Exploration: 通常探索の輝く宝箱は探索度100以上で抽選対象になる。探索度100到達は宝箱確定ではない。
- Battle: 通常探索の連戦待機は通常探索に適用する。表層は10秒、深部は15秒、深層/最深層/異界層は20秒。ボス挑戦と現在表示中の深部入口へ進む確認後の1戦は通常探索の連戦待機では止めないが、HP不足や宿屋直後の待機など開始条件は適用する。通常探索の深部/深層/最深層/異界層へ入場した時点で危険度を0%に戻し、入場済み階層は維持したまま次階層に向けて危険度を再蓄積する。深度目標Lvはエリア推奨Lvに対して深部+5、深層+15、最深層+25、異界層+40。深部以降の敵は、危険度倍率補正と深度目標Lvに対する種族・派生・役割ベースの敵ステータス生成式の高い方を戦闘時ステータスに使う。プレイヤー向けの通常ダンジョン目安はLvではなく、実際に出る通常敵（雑魚/やや強い）の戦力範囲で表示する。内部の推奨Lvは敵生成・深度補正・通常敵データがない表示のフォールバック用に残す。目安戦力はSP/MPを補助評価として控えめに扱い、AGIは最大35%まで緩やかに火力補正する。街道敵の通常EXPは、同Lv帯の通常敵に近い強さ相応のカーブで付与する。奥義と職業固有の従来必殺技は別系統で、奥義は装備スロット/継承ルール、従来必殺技は現在職の固有スキルとして扱う。
- Job arts: プレイヤー向け表記ではMPではなくSPを使う。SP回復/吸収を説明する奥義は、戦闘中に最大SP割合でSPを回復する。
- Drops: 通常戦の素材抽選は敵固有の素材ドロップを優先し、該当カテゴリがない場合は進行帯ごとのフォールバック素材プールから抽選する。地域素材カテゴリでは敵固有候補がある場合も都市の地域素材を低確率候補に残す。強化石・守護石系はこのフォールバックの enhance 枠にも含まれる。敵ドロップ武器・防具のうち `affix_enabled=true` のものは、ドロップ確定時に銘なし70%/能力銘20%/能力銘+種族銘8%/良品1.5%/逸品0.5%で銘抽選を行う。銘補正は個体に保存する。武器の種族特効はPvEの敵への直接ダメージ、防具の防護銘はPvEの敵から受ける直接ダメージにのみ適用し、PvP/チャンプ戦には適用しない。
- Valmon: 相棒ヴァルモンは通常探索勝利後にLv1で素材発見2%、Lv10で2.5%、Lv20で3%。Lv30から得意素材補正、Lv40から通常探索戦闘中1回だけ低確率追撃、Lv50から未発見要素の気配ヒント、Lv75からHP低下時の応急回復を1探索チェーン1回まで、Lv100で称号「名相棒」と特別ログ/プロフィール表示向け効果を解放する。
- Arena rank battle: 格上に勝った場合、勝者は相手の旧順位まで上がり、その間にいた冒険者は1つずつ下がる。格下に勝った場合、順位は変動しない。酒場NPC由来のNPCランカーも同じ順位帯に混ざり、NPCに勝った場合も同じ順位シフトを行う。TOP10はプレイヤー枠として保護し、当面は中級/一般NPCだけをプレイヤー順位帯より下の下位帯（最低51位以降）へ配置する。上級職NPCはランク外で待機し、ハヤト級でも表示Lv上限は50程度。NPCランカーは1日数回、11位以下を対象に自動ランク戦を行う。NPCは勝利ごとに表示Lvが1上がる（上限50）。legend枠と戦闘に向かないNPCは不出場。NPCの詳細ステータスはプレイヤー側では伏せる。
- Jobs/class change: 職業ランクは全職共通で10がマスター上限。必要職業EXPは職業階層で補正し、基本職1倍、中級職2倍、上級職5倍、伝説職10倍とする。未使用BPが残っている場合は転職できず、転職前にすべて能力へ割り振る必要がある。
- Job EXP rewards: 1回の報酬処理でプレイヤーに付与される職業EXPは最大3に抑える。探索深度、亜域、チャンプ戦などの補正後もこの上限を超えない。
- Equipment: 鍛冶強化は最大+5。+1ごとに基礎性能+3%を目安に伸び、低い基礎値では最低+強化値分を加算する。必要素材は+1/+2/+3/+4/+5で1/3/5/7/9個。武器は強化石、防具は守護石、装飾品は装飾強化石を使う。素材交換所では各欠片3個を対応する石1個へ精製できる。アクセ進化の秘境解放/秘境主撃破条件は、裏ダンジョンID 71〜74のみを対象にし、街道ID 75〜83は対象外とする。

## Public logs

- Loggable events: ランク戦で勝利して闘技場順位が上がった場合は全体チャットに番付上昇ログを流し、11位以下から10位以内に入った場合はTOP10入りログも流す。管理者は管理画面から管理人メッセージを全体チャットへ投稿できる。
- Equipment drops: 装備ドロップの公開ログはSSS/EPIC、または銘抽選で「逸品」になった装備を対象にする。通常のAランク装備獲得だけでは全体チャットへ流さない。
- Forbidden data: private user data, secrets, internal IDs unless safe.

## Balance

- Do not change numeric balance without explicit request.
- Enemy equipment drops: 通常探索の汎用装備ドロップは標準で武器1.5%、防具1%、装飾0.5%を目安にし、はじまりの草原はさらに低くする。敵固有装備ドロップも初期値の約半分に抑える。
- Champ battle rewards: チャンプ戦の素材報酬は勝敗に関わらず1〜2個に抑える。
- Champ battle level gap: チャンプ戦で挑戦者がチャンプよりLv10以上低い場合、1戦につき最大1回、レベル差に応じた確率でチャンプ最大HP基準の追加ダメージを与える。
- Champ battle skills: チャンプ戦では挑戦者とチャンプが現在職の職業固有必殺技をSP消費で発動できる。低レベル側はLv差2ごとに発動率+1%、最大+25%の補正を受ける。挑戦者本人のHP/SPは戦闘後に消費保存しない。
- Champ battle streak fatigue: チャンプは連勝数1ごとに戦闘時のATK/DEF/MAG/SPR/SPD/LUKが2%低下する。最大低下率は40%。HP共有値は低下させず、チャンプ交代時に連勝数0へ戻る。
- Enemy EXP: 砂都サンドラ以降は都市が進むほどEXP上昇を緩やかにする。現在の通常敵/ボスEXPはサンドラ94%、ルミナス88%、ネクロム82%、セレスティア76%、魔王城以降70%を目安に抑制する。
- Enemy hit rate: PvEで敵がプレイヤーを攻撃する場合は、回避だけで高難度を突破しにくくするため最低命中率を82%にする。魔王城ヴァルゼリア（city_id=10）以降の敵は最低命中率88%にする。
- Enemy stats: 魔王城ヴァルゼリア（city_id=10）の通常敵・ボスは、敵ステータス自動生成後の最終ステータスを1.1倍にする。
- When balance changes, update this file and related tests if present.
