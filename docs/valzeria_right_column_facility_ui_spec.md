# 冒険都市ヴァルゼリア メイン施設エリアUI仕様書

## ファイル名

valzeria_main_facility_area_ui_spec.md

---

## 1. 目的

本仕様書は、冒険都市ヴァルゼリアの中央から右側にあるメイン画面エリアへ、街の施設UIを実装するための仕様を定義するものである。

現在の画面では、上部に以下の主要ナビゲーションが存在している。

* 宿場町
* 迷宮へ
* 闘技場
* 武具屋
* 転職所
* ギルド
* 手紙箱
* 名簿

その下にあるメイン画面エリアは、現状プレースホルダーになっているため、この領域を「街の施設ハブ画面」として活用する。

本仕様の目的は、単なる空きスペースの埋め合わせではなく、以下の体験を成立させることである。

1. 上部ナビで街の場所を選ぶ
2. 選んだ場所に応じて中央メインエリアの内容が切り替わる
3. プレイヤーが施設カードから各機能へ進める
4. ヴァルゼリアの街にいる感覚を出す

---

## 2. 基本方針

### 2.1 メイン施設エリアの位置づけ

メイン施設エリアは、単なる説明表示領域ではなく、プレイヤーの行動を決める中心ハブとする。

役割は以下。

* 現在選択中の場所を表示する
* その場所に属する施設を表示する
* 利用可能な施設へ遷移させる
* 未解放施設や準備中施設を見せる
* 最新情報やおすすめ行動を表示する

### 2.2 なぜ中央エリアに施設を出すのか

施設を右カラムだけに詰め込むより、中央メインエリアを使う方が以下の点で優れている。

* 見やすい
* 施設数が増えても破綻しにくい
* スマホ対応しやすい
* 今後の拡張に強い
* プレイヤーが今どこにいるか分かりやすい

### 2.3 コンセプト

目指す形は以下。

```text
上部ナビで場所を選ぶ
↓
中央メインエリアに、その場所の施設カードが並ぶ
↓
施設カードから各機能へ進む
↓
下部ログで街の動きを確認する
```

## 4. メイン施設エリアの基本構成

メイン施設エリアは、以下の構成を基本とする。

```text
┌──────────────────────────────┐
│ 場所タイトル                  │
│ 短い説明文                    │
├──────────────────────────────┤
│ 主要施設カード                │
│ [施設] [施設] [施設]          │
│ [施設] [施設] [施設]          │
├──────────────────────────────┤
│ この場所の最新情報            │
│ ・おすすめ行動               │
│ ・未解放条件                 │
│ ・街の最近の動き             │
└──────────────────────────────┘
```

### 4.1 表示ブロック

1. タイトルエリア
2. 施設カード一覧
3. 補助情報エリア

### 4.2 タイトルエリア

表示内容は以下。

* 現在の場所名
* その場所の簡単な説明
* 必要に応じて現在の状況や目的

例。

```text
宿場町
冒険者たちが集まるヴァルゼリアの玄関口です。
```

## 15. 上部ナビとの連動

上部ナビを押したときに、メイン施設エリアの内容を切り替える。

### 15.1 実装方式

以下のどちらかで実装する。

#### 案1 ページ遷移方式

* /town
* /battle/areas
* /pvp
* /shop
* /jobs
* /guilds
* /messages
* /adventurers

#### 案2 タブ切り替え方式

* 同一ページ内でコンテンツのみ切り替える

### 15.2 推奨

初期実装ではページ遷移方式を推奨する。
理由は以下。

* 実装が分かりやすい
* URL管理しやすい
* Laravel構成と相性が良い
* 後から個別機能に拡張しやすい

## 20. 結論

現在の画面構成では、施設を右カラムに押し込むより、中央から右のメイン画面エリアへ実装する方が自然である。

この仕様では、上部ナビで場所を選び、その場所に属する施設カードをメイン画面エリアへ表示する。

これにより、以下を両立できる。

* 現代的で見やすい
