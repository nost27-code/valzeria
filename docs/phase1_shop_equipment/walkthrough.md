# フェーズ1 武具屋・装備 実装の完了

## 概要
`valzeria_phase1_shop_equipment_spec.md` に定義されていた「武具屋および装備機能」の実装が完了しました。
キャラクターは戦闘で獲得したGOLDを使って装備を購入し、ステータスを強化して、より強い敵に挑むことができるようになりました。

## 実装内容
1. **アイテム・装備DBと初期化**
   - `items` テーブルおよび `character_items` テーブルを作成しました。
   - `ItemSeeder` にて、木の剣や布の服、各種武器・防具・装飾品の計14種を登録しました。
   - 既存の全キャラクターに対して、初期装備（木の剣、布の服、魔除けの護符）を一斉付与するコマンドを実行しました。
   
2. **ステータス・戦闘の装備補正**
   - `CharacterStatusService` を新設し、素のステータスに装備品の補正を加算する処理を実装しました。
   - 左カラムのステータス表示が「装備補正込み」の最終ステータスとなり、現在装備しているアイテム一覧も表示されるようになりました。
   - 戦闘エンジン（`BattleService`）を根本的に改修し、戦闘時には素のステータスではなく**装備補正込みのステータス**を使ってダメージ計算・先攻後攻判定（STR, DEF, AGI等）を行うようにしました。

3. **武具屋と装備変更のUI**
   - 「武具屋（トップ・武器屋・防具屋・装飾屋）」の画面を作成しました（`ShopController`）。
   - GOLD不足やレベル不足、またはすでに所持している装備の場合は購入ボタンが無効化される（エラー表示になる）チェック処理を追加しました。
   - 「装備変更」画面を作成し、現在所持している装備一覧から簡単に着脱（別装備との入れ替え）ができるようにしました（`EquipmentController`）。

## 確認・テスト手順
1. 左カラムのステータスが、初期装備（木の剣など）の補正を受けた値（STR10+3=13 など）になっているかを確認してください。
2. 闘技場や迷宮で戦闘を行い、与えるダメージが装備によって底上げされていること、またGOLDを獲得できることを確認してください。
3. メニューの「武具屋」から「武器屋」などへ入り、新しい装備（例：鉄の剣 120G）を購入してみてください。
   ※GOLDが足りない場合やレベルが足りない場合は購入できません。
4. 購入後、左カラムの「装備変更」リンクから装備変更画面を開き、新しく購入した装備を「装備する」で付け替えてみてください。
5. 装備変更後、左カラムの現在の装備一覧およびステータス（STRなど）がリアルタイムに変化し、再び戦闘した際にダメージが上昇しているかを確認してください。
