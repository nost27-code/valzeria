# ステータス計算の修正およびボスドロップの設定計画

現在発生している「倍率補正によるマイナス」や「ボスの印ドロップがない」問題を解決するため、以下の修正を行います。

## 1. 職業の「倍率補正」の廃止
ご指摘の通り、レベルが上がりステータス基礎値が高くなると「1.3倍」や「0.9倍」といった倍率補正の影響が大きくなりすぎ、装備の数値を上書きしてしまう問題が発生します。
そのため、`CharacterStatusService` のステータス計算式から **「職業による倍率補正（×1.3 等）」の計算を完全に削除** します。
今後は「基礎値 ＋ レベルに応じた固定値ボーナス ＋ 装備ボーナス」のみをステータスとして加算するようにします。

## 2. 職業パラメータの一括アップデート
ご提示いただいた最新の「Job設定マトリクス表」の内容に基づき、全職業のマスターボーナス（HP、MP、ATK等）の値をデータベース初期化ファイル（`JobSystemSeeder`）へ一括で適用します。
これにより、戦士のATKボーナスも「12」として正しく計算されるようになります。

## 3. ボスの「印」ドロップ設定の追加
データベースを確認したところ、各ボスの「印」アイテム自体は存在しているものの、ボスの「ドロップアイテム」として紐づいていない状態（Null）でした。
すべてのボスに対して、対応する「〇〇の刻印（または王印）」をドロップするように設定を修正します。
