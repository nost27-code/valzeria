# 職業ランクに応じたステータスボーナス実装計画

## 目的
現在の職業のランク（レベル、★1〜★10）に応じて、職業ごとに設定された特徴的なステータスボーナスが加算される仕組みを実装します。これにより、同じ職業でもランクが高いほど強力になり、やり込み要素や戦闘での個性が強化されます。

## 実装方針（アプローチ）
1. **ランク情報の活用**:
   - すでに実装されている職業の熟練度（`job_histories.job_level`、最大10）を「★の数（ランク）」としてそのまま活用します。
2. **ボーナス値の定義**:
   - 既存の `job_classes` テーブルには `bonus_hp` や `bonus_str` などのカラムが存在します（例：戦士は `bonus_hp` = 20）。
   - これを「1ランクあたりのボーナス値」と定義します。
     - *例: 戦士の `bonus_hp` が 20 で、現在のランクが★5なら、20 × 5 = **HP+100** のボーナスがつく。*
3. **加算処理の実装箇所**:
   - `CharacterStatusService` の `getFinalStats()` メソッドを修正します。
   - 現在の職業とそのランクを取得し、計算したボーナス値を「装備品のボーナス」と同じ扱い（加算値）として足し合わせます。
   - これにより、ステータス画面などの UI 上でも自動的に「緑色の文字（+XXX）」として表示されるようになります。

## 変更対象ファイル
### `app/Services/CharacterStatusService.php`
- `getFinalStats()` メソッド内に、職業ランクによるボーナス計算処理を追加。
- 装備品によるボーナス値の初期値に、職業によるボーナス値を代入したうえで、装備ボーナスを加算する形に改修。
