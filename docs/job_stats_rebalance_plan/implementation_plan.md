# 職業成長倍率（全8項目）再調整の計画

全職業の成長倍率を、これまでの「主要5項目」から「全8項目（HP, MP, ATK, DEF, MAG, SPR, SPD, LUK）」へ拡張し、ゲームバランスを考慮したマイルドな配分に再設定します。

## ルールと方針

1. **実質的な強さの統一（総合値の統一）**
   - 同じランクの職業は、全8項目の合計倍率を（ほぼ）統一します。
     - 基本職 (Normal): 合計 **840%** (平均 105%)
     - 中級職 (Middle): 合計 **1000%** (平均 125%)
     - 上級職 (Advanced): 合計 **1200%** (平均 150%)
     - 伝説職 (Legend): 合計 **1440%** (平均 180%)
2. **尖りすぎないマイルドな配分**
   - 「戦士の魔法は50%」といった極端に使えないステータスは作りません。
   - 不得意なステータスであっても、最低 80%〜100% 程度は確保し「人間としての最低限の成長」を担保します。
3. **最大倍率の制限**
   - どんなに高くても最大倍率は **225%** とし、一部の伝説職や上級職の得意分野にのみ適用します。

## 実装ステップ

1. ユーザーの承認を得る。
2. `JobSystemSeeder.php` のステータス設定配列を更新する（MP, SPR, LUKも追加）。
3. コマンド `php artisan db:seed --class=JobSystemSeeder` を実行し、データベースを更新する。
