# 伝説職解放用裏ボス・アイテム追加および転職条件の実装計画

ユーザーのご提案に基づき、伝説職を解放するための専用アイテムをドロップする「裏ボス」と、そのボスが配置される「裏ダンジョン」の追加、および段階的な実装方針（フェーズ1〜3）を計画しました。

## 概要

伝説職を解放するためには、「特定の職業のマスター」だけでなく、「裏ボスの撃破」によって得られる専用アイテムが必要となる仕様を実装します。

## フェーズ1: モンスター定義・エリア・アイテム・ドロップの追加

各Seederを更新し、必要な基礎データを追加します。

1. **アイテムの追加 (`ItemSeeder.php`)**
   - 英雄の証
   - 深淵の核
   - 古代核
   - 竜神の鱗
   - 時空結晶

2. **エリア（裏ダンジョン）の追加 (`AllDungeonsSeeder.php`)**
   - 深淵の裂け目（魔界都市ネクロム）
   - 古代錬成炉（鉄鋼都市グランベルグ）
   - 竜王の聖域（天空都市セレスティア）
   - 次元回廊中枢（魔導都市ルミナス） ※既存の次元回廊とは別の裏エリアとして設定
   ※「魔王城ヴァルゼリア」は既存の「終焉の祭壇」をそのまま利用

3. **裏ボスの追加 (`BossSeeder.php` への追加)**
   - 魔王ヴァルゼリア (※既存IDを利用し、ドロップのみ追加設定)
   - 深淵王アビスロード
   - 錬成炉守護者オルディオン
   - 竜王バハムート
   - 次元回廊の支配者クロノアーク

4. **ドロップ定義の追加 (`EnemyDropsSeeder.php`)**
   - 各ボスに対応する「専用アイテム」を、初回撃破100%、再戦時3〜5%の確率でドロップするように設定します。

## フェーズ2: 伝説職の解放条件に必要アイテムを追加

`JobSystemSeeder.php` の処理を拡張し、伝説職の `JobRequirement` としてアイテム要求を追加します。

- ヴァルゼリアの英雄 ＝ 勇者＋剣聖＋大賢者 ＋ `英雄の証`
- 深淵歩き ＝ 暗黒騎士＋幻影王＋大賢者 ＋ `深淵の核`
- 古代錬成王 ＝ 黄金商人＋機工王＋賢商王 ＋ `古代核`
- 竜神 ＝ 竜騎士＋武神＋勇者 ＋ `竜神の鱗`
- 時空王 ＝ 大賢者＋幻影王＋機工王 ＋ `時空結晶`

※ 現在の `job_requirements` テーブルは `requirement_type = 'item'` をサポートしているため、そのまま適用可能です。

## フェーズ3: 専用ボス（ダンジョン）への挑戦条件を追加

「裏ボスには、必要な上級職をマスターしていなければ挑戦できない」という仕組みを実装します。

1. **マイグレーションの作成**
   - `areas` テーブルに `required_master_job_keys` (JSON型、nullable) といったカラムを追加し、エリア入場に必要なマスター済み職業を指定できるようにします。
   （例：深淵の裂け目への入場には `['dark_knight', 'phantom_king', 'grand_sage']` を要求）
2. **モデルへのアクセス判定追加**
   - `Area` モデルに `$area->isUnlockedFor($character)` のような判定メソッドを拡張し、指定された職業をすべてマスターしているかどうかを確認します。
3. **UIへの反映**
   - 条件を満たしていない場合、街のダンジョン一覧にて入場不可（ロック状態）として表示します。

> [!NOTE]
> 既存の「魔王城中枢（終焉の祭壇）」で戦う通常の『魔王ヴァルゼリア』についても、初回撃破で「英雄の証」を落とすように設定します。もし魔王だけは職業マスター関係なく挑戦可能とする（ストーリー進行ボスであるため）のであれば、フェーズ3の挑戦条件制限は「魔王以外の4体の裏ダンジョン」にのみ適用することとします。
