# 防具マスタのバランス調整とソート

## 目標
1. **マスタのソート**: `docs/armor_master.tsv` を、武器の時と同じルール（店売り品のランク順 → 合成品のランク順・価格順 → ドロップ品のランク順・価格順 → その他）でソートします。
2. **店売り価格の適正化**: 各街の店売り防具の価格を、その街の周辺に出現する「雑魚モンスターの平均ドロップ金額 × 約42.5倍」に再設定します。
3. **合成品の強化**: 合成品（素材を用いて作成される防具）の基本防御力等を、同ランクの店売り品の「1.3倍前後」に強化します。また、防具の種別（重装、軽装、法衣、外套など）に応じて、防御力以外のサブステータス（HP, MP, STR, MAG, AGI, SPR, LUK等）にボーナスを付与し、合成する価値を出します。

## 提案する変更内容

### [MODIFY] docs/armor_master.tsv
以下の手順でデータを更新・書き換える専用のPHPスクリプトを作成・実行します。
1. **価格（Price）の再計算**: `docs/enemy_master.tsv` から各街（エリア）のボスを除く平均ドロップ金額を算出し、`unlock_city_id` に応じて店売り品の価格を `平均金額 × 42.5` とします。合成品やドロップ品はランクをベースにした基準価格に設定します。売値はその半分とします。
2. **ステータスの再計算（合成品等）**:
   - 価格から内部的なランク（K〜SSS）を判定します。
   - 店売り品の基準防御力（Base DEF）をランクごとに定義します。
   - **合成品** の基本防御力を「店売り品の約1.3倍」に設定します。
   - さらに防具のサブタイプ（重装、軽装、法衣など）に応じてサブステータス（HP, AGI, SPRなど）を付与します。
     - **重装**: 高いDEF、HP増加、AGI微減
     - **軽装**: 標準DEF、AGI微増
     - **法衣**: やや低いDEF、高いSPRとMP増加
     - **外套**: 中程度のDEF、AGIとLUK増加
     - **革鎧**: やや高いDEF、AGI微増
     - **旅装**: 標準DEF、LUKとHP微増
     - **魔装**: 中程度のDEF、MAGとSPR増加
3. **ソートの実行**: 上記の価格と入手区分（店売り、ドロップ、合成品）に基づいて、指定のルール通りにTSV内の行順を並び替えます。
