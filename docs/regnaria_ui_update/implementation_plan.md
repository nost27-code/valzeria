# 施設カードの画像背景化 実装計画

絵文字の透かしではなく、実際のイラスト画像を施設カードの背景として薄く敷き詰める改修を行います。

## User Review Required

> [!IMPORTANT]
> 施設は全部で約25個（各タブの合計）あります。すべてを一度に画像生成すると非常に時間がかかるため、**まずは「宿場町」の6施設（宿屋、訓練所、案内所、銀行、酒場、教会）の画像を生成し、UIに組み込んで雰囲気を確かめていただく**というステップを踏みたいと考えています。
> 問題なければ、以下のテイストで画像を生成します。
> 
> **画像テイストの希望について**
> 以下からお好みのスタイルをお選びいただけますか？（特に指定がなければ 1 で進めます）
> 1. ファンタジーRPG風の美しいイラスト（水彩調など）
> 2. レトロゲーム風のドット絵（ピクセルアート）
> 3. ダークファンタジー風の重厚なイラスト

## Proposed Changes

### 1. 施設背景画像の生成と保存
- `generate_image` ツールを使用し、宿場町の各施設（宿屋、訓練所、案内所、銀行、酒場、教会）を表現する背景画像を生成します。
- 生成した画像を `public/images/facilities/` ディレクトリに保存します（例：`inn_bg.png` など）。

### 2. データ構造の改修
- `app/Livewire/MainScreen.php` の `town` のデータ配列に、`bg_image` プロパティを追加し、画像のパス（例: `images/facilities/inn_bg.png`）を持たせます。

### 3. UI（ブレードテンプレート）の改修
- `main-screen.blade.php` 内の施設カード表示部分を改修します。
- 先ほど追加した「絵文字の透かし」を削除します。
- カード自体の背景画像として `style="background-image: url('{{ asset($facility['bg_image']) }}');"` を設定し、`bg-cover` `bg-center` クラスを付与します。
- そのままでは文字が読めなくなるため、画像の上に白色または黒色の半透明なオーバーレイ（`bg-white/80` や `bg-slate-900/60` など）を重ね、テキストの視認性を確保します。

## Verification Plan
- 宿場町のタブを開いた際、各施設カードの背景に生成されたイラストが薄く表示されているか。
- 画像の上に表示されるテキストがしっかりと読める状態になっているか。
- 未解放や準備中のカードのグレーアウト処理が、背景画像にも正しく適用されているか。
