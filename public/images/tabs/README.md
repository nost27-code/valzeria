# Tab Icons

Place navigation tab icons here as WebP files.

- `tab_move.webp` - 移動
- `tab_town.webp` - 街
- `tab_dungeon.webp` - 探索
- `tab_colosseum.webp` - 闘技場
- `tab_temple.webp` - 神殿
- `tab_guild.webp` - ギルド
- `tab_message.webp` - 手紙
- `tab_settings.webp` - 設定

Recommended size: 64x64px or 128x128px, transparent background.
